# Ozzie Young — Sailing Portfolio

A one-page sailing portfolio for Ozzie Young, Class of 2028.

## Setup

### 1. Add your media files

You need to add photos and videos to the `public/` folder before the site will look right.

**Videos — copy these files:**
```
public/videos/hero.mov        ← IMG_5252.mov (the hero video)
public/videos/highlight.mov   ← the second video (highlight reel)
```

**Photos — rename and copy your 13 photos:**
```
public/photos/photo1.jpg   ← 83F44E59 (poster/action shot)
public/photos/photo2.jpg   ← 1418D169 (racing shot for About section)
```

### 2. Install and run

```bash
npm install
npm run dev
```

Open http://localhost:5173 to see the site.

### 3. Deploy

```bash
git add .
git commit -m "your message"
git push
```

Vercel auto-deploys on every push to main.

## Tech stack

- React (via Vite)
- No dependencies beyond React
- Google Fonts: Bebas Neue + Inter
